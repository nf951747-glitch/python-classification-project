# Python Data Classification Project

A machine learning classification project developed in Python using the Iris dataset. The project demonstrates the complete machine learning workflow, including data loading, preprocessing, model training, evaluation, and performance analysis using the K-Nearest Neighbors (KNN) algorithm.

---

## Features

- Load and preprocess the Iris dataset
- Split data into training and testing sets
- Standardize features
- Find the best value of K automatically
- Train a K-Nearest Neighbors (KNN) classifier
- Evaluate model performance
- Generate a classification report
- Display the confusion matrix

---

## Technologies Used

- Python
- NumPy
- Pandas
- Scikit-learn

## Installation

Clone the repository

```bash
git clone https://github.com/nf951747-glitch/python-classification-project.git
```

Move to the project folder

```bash
cd python-classification-project
```

Install the required libraries

```bash
pip install -r requirements.txt
```

Run the project

```bash
cd src
python main.py
```

---

## Workflow

1. Load the Iris dataset
2. Preprocess and scale the data
3. Split the dataset into training and testing sets
4. Determine the optimal K value
5. Train the KNN classifier
6. Evaluate the trained model
7. Display performance metrics and the confusion matrix

---

## Results

The trained model achieved approximately:

- **Accuracy:** 93.33%
- **Macro F1 Score:** 93.27%

The output also includes:

- Classification Report
- Confusion Matrix
- Best K Value
